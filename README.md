## üì¶ Getting Started

### Prerequisites
- Node.js 18+ and pnpm
- Environment variables (see `.env` file)

### Installation

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

Visit `http://localhost:3000` to see the application.
